Parity Checking

It is a simple error detection technique used in transmission of digital data. It ensures accurate data transmission and helps to identify and remove errors caused by factors such as noise.

How it works

A parity bit is appended to the original data bits to create data with an even or odd number of 1 bits. The sender then sends this data to the receiver.

At the receiver end, the number of 1s are checked. The data is considered accurate if the number(even or odd) matches the data transmitted by the sender.

Even Parity

The total number of bits in the received data is expected to be even. Hence if there is an even number of 1s in the data to be transmitted, the parity bit 0 is added; if there is an odd number of 1s, parity bit 1 is added.

Odd Parity

Opposite to even parity: the total number of bits in the received data is supposed to be odd. If there is an odd number of 1s in data to be transmitted, parity bit 0 is added; and if there is an even number of 1s, parity bit 1 is added.
